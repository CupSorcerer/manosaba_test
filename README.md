# 魔法少女的魔女审判 - 角色性格测试

## 项目简介

这是一个基于《魔法少女的魔女审判》游戏的MBTI性格测试项目，通过回答一系列性格测试题，判断用户最像游戏中的哪个角色。

## 功能特点

- **哥特式暗黑风格**：采用游戏原版的视觉风格，深红色调与黑色背景营造神秘氛围
- **14个角色匹配**：涵盖游戏中所有主要角色，每个角色都有详细的性格分析
- **20道测试题**：精心设计的MBTI性格测试题，涵盖四个维度（E/I, S/N, T/F, J/P）
- **角色立绘展示**：每个角色都有精美的立绘展示
- **详细结果分析**：显示角色性格描述、特质标签、魔法能力和MBTI类型分析
- **响应式设计**：适配不同屏幕尺寸，支持移动端访问
- **分享功能**：可分享测试结果到社交媒体
- **背景音乐**：自动循环播放游戏风格BGM，支持淡入淡出效果，可一键静音
- **角色主题色**：每个角色都有独特的代表色，结果页面会根据匹配角色动态更换主题色调

## 如何使用

网址：https://cupsorcerer.github.io/manosaba_test/

### 本地运行

1. 确保你的电脑上安装了Python
2. 打开命令行终端，进入项目目录
3. 运行以下命令启动本地服务器：

   ```bash
   python -m http.server 8080
   ```

4. 在浏览器中访问 `http://localhost:8080` 开始测试

### 测试流程

1. 点击首页的「开始测试」按钮
2. 回答20道性格测试题（每题4个选项）
3. 测试完成后，系统会根据你的回答匹配最适合的角色
4. 查看详细的结果分析，包括角色立绘、性格描述、特质标签等
5. 可以选择「重新测试」或「分享结果」

## 技术实现

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **样式设计**：基于游戏原版哥特式风格，使用CSS渐变、阴影和动画效果
- **数据处理**：客户端JavaScript处理测试逻辑和角色匹配算法
- **响应式设计**：使用CSS媒体查询适配不同设备
- **音频控制**：使用Web Audio API实现BGM的自动播放、淡入淡出和静音控制
- **动态主题**：通过CSS变量和JavaScript动态计算颜色变体，实现角色专属主题色

## 自定义扩展

### 添加新角色

1. 在 `script.js` 文件的 `characters` 对象中添加新角色数据
2. 确保新角色包含以下属性：
   - `name`：角色名称
   - `emoji`：角色表情符号
   - `image`：角色立绘路径
   - `mbti`：MBTI类型
   - `desc`：角色描述
   - `traits`：性格特质数组
   - `magic`：魔法能力
   - `scores`：MBTI维度得分
3. 将角色立绘图片添加到 `image` 文件夹

### 修改测试题

在 `script.js` 文件的 `questions` 数组中修改或添加测试题，每个题目包含：
- `question`：问题内容
- `options`：选项数组，每个选项包含 `text`（选项文本）和 `scores`（得分）

### 修改角色代表色

在 `color.txt` 文件中修改角色代表色，格式为 `角色ID:十六进制颜色码`，例如：
```
ema:#ffc0cb
hiro:#ff0000
coco:#b24f36
```
修改后结果页面会自动应用新的主题色调。
